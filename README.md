# BPS-Project

**Title: Evaluating User Satisfaction with Low-Code Enabled Business Process Standardization: An Empirical Study in a Medium Enterprise**

This project was in collaboration with Frutta - Corporate Refreshment & Experience Specialist where I was in charge of the Business Product Standardization using Low Code or No Code Tools like Google Web App and Google App Script. This project measures the User Satisfaction of the employees using different models after the fact.

## Tools Used:
- JASP for performing Confirmatory Factor Analysis (CFA)
- SmartPLS for performing Partial Least Squares Structural Equation Modeling (PLS-SEM), tests of Reliability and Validity

## What I learned:
- Metrics required to measure reliability and validity
- Hypothesis testing using PLS-SEM
- Mediation Analysis using Bootstrapping
